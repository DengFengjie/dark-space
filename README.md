# 深空探测可视化展示平台

一个基于B/S架构的专业级深空探测三维可视化平台，使用Vue 3、Vue Router和Three.js构建，提供沉浸式的宇宙探索体验。

## 🌟 功能特性

### 核心功能
- 🌌 **专业首页** - 精美的产品介绍页面，包含功能展示和快速导航
- ☀️ **太阳系全景** - 完整展示八大行星及其轨道系统
- 🔴 **火星探测** - 详细展示火星及其卫星（火卫一、火卫二）
- 🌙 **地月系统** - 展现地球与月球的引力互动关系
- 🎮 **交互控制** - 支持旋转、缩放、平移等多维度视角控制
- 📊 **信息面板** - 实时显示天体参数和详细数据
- ✨ **视觉效果** - 基于物理的光照渲染和星空背景

### 页面路由
- `/` - 精美首页介绍
- `/solar-system` - 太阳系三维展示
- `/mars` - 火星探测展示
- `/moon` - 地月系统展示

## 🛠️ 技术栈

### 前端技术
- **Vue 3** - 现代化渐进式JavaScript框架
- **Vue Router** - 官方路由管理器
- **Three.js** - WebGL 3D图形渲染引擎
- **Vite** - 下一代前端构建工具
- **JavaScript ES6+** - 现代JavaScript语法

### 后端技术
- **Node.js** - JavaScript运行时环境
- **Express** - 轻量级Web应用框架

## 📦 安装与运行

### 1. 安装依赖

```bash
npm install
```

### 2. 开发模式运行

```bash
npm run dev
```

访问 http://localhost:3000

### 3. 生产构建

```bash
npm run build
```

构建后的文件将输出到 `dist` 目录。

### 4. 启动生产服务器

```bash
npm run server
```

访问 http://localhost:8080

## 📖 使用说明

### 首页导航
1. **浏览功能介绍** - 滚动查看平台的各项功能特性
2. **快速跳转** - 点击导航栏链接或直接点击功能卡片
3. **开始探索** - 点击"开始探索"按钮进入三维展示页面

### 三维场景控制
- **左键拖拽** - 旋转视角，观察天体不同角度
- **右键拖拽** - 平移视角，移动观察位置
- **滚轮缩放** - 放大或缩小视图
- **重置视角** - 恢复到初始观察角度

### 显示控制
- **显示/隐藏轨道** - 切换行星轨道线显示
- **暂停/恢复旋转** - 控制天体公转和自转动画

### 页面切换
在每个三维展示页面顶部都有导航栏，可以快速切换到其他展示场景。

## 🎨 核心功能实现

### 3D场景渲染
- 使用Three.js创建高性能3D场景
- SphereGeometry球体几何体表示天体
- MeshPhongMaterial材质实现真实光照效果
- PointLight点光源模拟恒星光照

### 天体运动模拟
- 椭圆轨道数学计算
- 行星公转动画实现
- 天体自转效果
- 多卫星系统模拟

### 路由管理
- Vue Router实现SPA单页应用
- 页面间平滑过渡动画
- 路由级别代码分割

### 交互控制
- OrbitControls轨道控制器
- 鼠标事件精确处理
- 相机视角动态管理

### 响应式设计
- 适配不同屏幕尺寸
- 移动端友好界面
- 流畅的动画效果

## 🔌 API接口

后端提供RESTful API接口：

- `GET /api/planets` - 获取行星列表信息
- `GET /api/info/:planet` - 获取特定天体详细信息
