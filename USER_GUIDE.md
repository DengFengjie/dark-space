## 系统要求

### 硬件要求
- **CPU**: Intel i5 / AMD Ryzen 5 或更高
- **内存**: 4GB RAM 最低，8GB 推荐
- **显卡**: 支持WebGL 2.0的显卡
- **硬盘**: 500MB 可用空间

### 软件要求
- **操作系统**: Windows 10+, macOS 10.15+, Linux (Ubuntu 18.04+)
- **Node.js**: 版本 16.0 或更高
- **浏览器**: 
  - Chrome 90+ ✅ 强烈推荐
  - Firefox 88+ ✅
  - Safari 14+ ✅
  - Edge 90+ ✅

## 安装步骤

### 第一步：安装Node.js

如果尚未安装Node.js：

1. 访问 [https://nodejs.org/](https://nodejs.org/)
2. 下载并安装LTS版本
3. 验证安装：
   ```bash
   node --version
   npm --version
   ```

### 第二步：获取项目文件

通过以下方式之一获取：

**方式A：克隆仓库**
```bash
git clone <repository-url>
cd dark-space
```

**方式B：解压项目包**
```bash
unzip dark-space.zip
cd dark-space
```

### 第三步：安装依赖

```bash
npm install
```

这将安装所有必需的依赖包（可能需要几分钟）。

## 快速启动

### 方法一：使用启动脚本（最简单）

#### Windows用户
双击运行 `start.bat` 文件

#### Mac/Linux用户
```bash
chmod +x start.sh
./start.sh
```

脚本会自动：
1. 检查Node.js是否安装
2. 安装依赖（如果需要）
3. 让您选择运行模式

### 方法二：命令行启动

#### 开发模式（推荐用于学习和开发）
```bash
npm run dev
```
- 地址: http://localhost:3000
- 支持热重载
- 修改代码自动刷新

#### 生产模式（推荐用于演示和部署）
```bash
# 构建
npm run build

# 启动服务器
npm run server
```
- 地址: http://localhost:8080
- 性能优化
- 适合长期使用

## 常见问题

### ❓ 页面显示空白

**原因1**: Node.js未安装
```bash
# 检查是否安装
node --version

# 如果未安装，请下载安装
# https://nodejs.org/
```

**原因2**: 依赖未安装
```bash
# 重新安装依赖
rm -rf node_modules
npm install
```

**原因3**: 端口被占用
```bash
# 修改vite.config.js中的端口
server: {
  port: 3001  # 改为其他端口
}
```

### ❓ 3D场景加载缓慢

**解决方案**：
1. 关闭浏览器其他标签页
2. 更新显卡驱动
3. 降低浏览器硬件加速
4. 使用Chrome浏览器（性能最优）

### ❓ 鼠标控制不流畅

**解决方案**：
1. 检查浏览器是否为最新版本
2. 清除浏览器缓存
3. 尝试禁用浏览器扩展

### ❓ 如何修改天体数据？

编辑文件：`src/data/celestialData.js`

```javascript
// 示例：修改火星颜色
mars: {
  radius: 5,
  color: 0xFF0000,  // 改为这个颜色
  // ...
}
```

保存后，开发模式会自动刷新。

### ❓ 如何添加新天体？

1. 在 `celestialData.js` 添加数据
2. 在视图文件中创建天体对象
3. 添加到场景

参考现有代码结构即可。

### ❓ 如何部署到服务器？

```bash
# 1. 构建
npm run build

# 2. 将dist目录上传到服务器
# 3. 配置Web服务器（Nginx/Apache）指向dist目录
```

或使用Node.js服务器：
```bash
# 在服务器上运行
npm run server
```

## 🎓 进阶使用

### 自定义样式

修改全局样式：`src/styles/global.css`

```css
:root {
  --primary-color: #64a3ff;  /* 修改主题色 */
  --accent-color: #FFD700;   /* 修改强调色 */
}
```

### 添加新页面

1. 创建组件：`src/views/NewPage.vue`
2. 添加路由：`src/main.js`
3. 添加导航入口

### API扩展

在 `server/index.js` 中添加新接口：

```javascript
app.get('/api/new-feature', (req, res) => {
  res.json({ data: '...' })
})
```

### 性能优化

**前端**：
- 使用懒加载
- 减少多边形数量
- 优化纹理大小

**后端**：
- 启用Gzip压缩
- 使用CDN
- 实现缓存策略
